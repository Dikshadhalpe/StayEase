# 🏠 StayEase – Mini Airbnb Inspired Web Application

StayEase is a mini full-stack web application inspired by Airbnb, developed with limited yet core features. It enables users to browse listings, view property details, and manage listings through a clean and responsive interface and aslo adding reviews.

This project demonstrates hands-on experience with backend APIs, database integration, and RESTful architecture.

## ▶️ How to Run This Project Locally

### 1️⃣ Clone the Repository

```bash id="m19xwq"
git clone https://github.com/dikshadhalpe/StayEase.git
```

### 2️⃣ Navigate to Project Folder

```bash id="z82kdn"
cd StayEase
```

### 3️⃣ Install Dependencies

```bash id="q92kdm"
npm install
```

### 4️⃣ Setup Environment Variables

Create a `.env` file in the root directory and add:

```id="p92ksm"
PORT=8000
MONGO_URI=your_mongodb_connection_string
```

### 5️⃣ Run the Application

```bash id="c81kxp"
npm start
```

Open in browser:

```id="b28xmk"
http://localhost:8000
```
